# CS288

The course covers Linux programming with Apache Web and MySql database using Php/Python and C as primary languages. 
It consists of four stages: basic tools such as Bash and C programming; searching trees and matrix computing, end-to-end 
applications such as one that constantly presents top 100 stocks; and extending the applications to run on multiple machines. 
The course provides students with hands-on experience for programming relatively large applications.
